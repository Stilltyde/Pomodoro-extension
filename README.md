# Pomodoro-extension
Плагин создан с целью оптимизации рабочего процесса посредством разбиения всего рабочего времемни на отрезки. Работа плагина основана на популярном методе pomodoro который предпологает разделение работы на периоды по 25 минут, между которыми предлагаетяс делать пятиминутные перерывы. Считается, что подобная методика улучшает эффективность и концентрацию.
