# Pomodoro-timer
Плагин создан с целью оптимизации рабочего процесса посредством разбиения всего рабочего времемни на отрезки. Работа плагина основана на популярном методе pomodoro который предпологает разделение работы на периоды по 25 минут, между которыми предлагаетяс делать пятиминутные перерывы. Считается, что подобная методика улучшает эффективность и концентрацию.

##Использование
- Запуск таймера
**Ctrl + P** -> **>Pomodoro start**
Оповещение об успешном запуске таймер появится в правом нижнем углу. 

- Остановка таймера
**Ctrl + P** ->**>Pomodoro stop**

Плагин был создан с использованием Yo Code,  это помогло избежать ненужных временных затрат на построение основной структуры расширения, и с помощью Javascript, который использовался как основной язык программирования. Сделан Сосонкиным Артёмом Владимировичем М3107.
